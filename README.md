# Java Calculator

This is a simple console-based Calculator program written in Java.  
It allows the user to perform basic arithmetic operations such as:

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

The program takes two numbers as input from the user and then performs the selected operation using a switch-case structure.
